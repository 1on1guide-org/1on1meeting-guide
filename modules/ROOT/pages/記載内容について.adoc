タイトル:: 
記載内容を簡潔に表す言葉やメタファー [.small]#<<note-構成-kome1>>#

[#note-構成-kome1, reftext=※1]
[.small]#※1{xref-用語集-メタファー}#

はじめに:: 
サブタイトル的に内容を推測できるものとして、
どんな状況でどんなことをしてどんな価値を得るのかを記載しています。

// ========================================
// ライター向けの記載内容
ifdef::convert-for-writers[]
[WARNING]
.[.orange.big]#*【編集者向け情報】 的確に表した言葉を選んでください*#
====
====
endif::convert-for-writers[]
// ========================================

要約:: 
使用例を除く詳細をまとめ理解を促すものとして、
「タイトル」や「はじめに」ではわからない内容部分について簡潔にわかりやすく交えた説明をしています。

// ========================================
// ライター向けの記載内容
ifdef::convert-for-writers[]
[WARNING]
.[.orange.big]#*【編集者向け情報】 状況や問題、フォースをも踏まえつつ簡潔にわかりやすい説明を記載してください*#
====
====
endif::convert-for-writers[]
// ========================================

状況:: 
パターンを適用するべき状況を表すものとして、{name-1on1}において対応が必要だと感じる顕在化した状況や{name-1on1}上で感じる小さな問題や課題を記載しています。

問題:: 
上記状況を放置することで起こる問題もしくは、すでに上記状況に陥ったことで顕在化している問題として、{name-1on1}以外で起こる{name-1on1}で解決に向かえる問題を記載しています。

// ========================================
// ライター向けの記載内容
ifdef::convert-for-writers[]
[WARNING]
.[.orange.big]#*【編集者向け情報】 状況が悪化して起こる問題に着目して記載してください*#
====
====
endif::convert-for-writers[]
// ========================================

フォース:: 
問題に至る背景として、問題が起きるほどの状況を生み出してもなお対応を躊躇う理由や状況を生み出す暗黙的な価値観や行動を記載しています。[.small]#<<note-構成-kome2>>#

[#note-構成-kome2, reftext=※2]
[.small]#※2{xref-用語集-フォース}#

解決方法:: 
適用を行う際の実際の抽象的な振る舞いを記載しています。

結果状況:: 
適用が上手くいったのちに現れる状況や解決方法が成功した場合のリターンについて記載しています。

// ========================================
// ライター向けの記載内容
ifdef::convert-for-writers[]
[WARNING]
.[.orange.big]#*【編集者向け情報】 要約や問題の裏返しではなく、その先に進んでいる内容を記載してください*#
====
====
endif::convert-for-writers[]
// ========================================

使用例:: 
パターンを適用するべきタイミングや背景について、実際の状況などが思い浮かぶ程度に具体的な内容を記載しています。
内容についてはより具体的でありつつ、読者が自分なりの適用方法をその都度考えられるように大きな余地を残すように記載されています。

関連パターン:: 
関連パターンは大きく分けて以下の二つがあります。
* 当該パターンを補強したり補完する別のパターン
* 当該パターンの逆のパターンなどを含む副作用・副反応を避けるために考慮すべきパターン
